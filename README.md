# Data Analyst Task

Welcome to my submission for the **Data Analyst Task** at *My Thriving Child*. This project demonstrates my ability to analyze social media performance data, extract actionable insights, and build a visually engaging dashboard. It also includes Python section to showcase data handling and visualization skills in Pandas.

## Overview

This task is divided into two parts:

### Part 1: Instagram Performance Dashboard

Using the provided dataset, I created a dashboard to visualize key social media metrics, focusing on engagement, reach, and content effectiveness.

#### Deliverables:
- An interactive dashboard created using PowerBI
- Relevant visualizations (line charts, bar graphs, KPIs)
- A brief written report explaining key insights

#### Questions Addressed:
1. What key metrics are best for measuring social media performance?
2. How can we evaluate the success of Instagram Reels?
3. How to design A/B tests for Facebook creatives?
4. What insights emerge from the dataset?
5. What additional analyses could further improve understanding?


### Part 2: Facebbok Performance Dashboard

The Python task demonstrates my ability to clean and analyze data programmatically in pandas.

####  Tasks Completed:
- **Data Cleaning**: Handled missing/inconsistent values using Pandas.
- **Basic Analysis**: Calculated average engagement rate; identified top-performing posts.
- **Visualizations**: 
  - Line chart showing engagement over time
  - Bar chart comparing Reels vs. Static Posts
- **Bonus**: Implemented a simple function to predict post performance based on thresholds
